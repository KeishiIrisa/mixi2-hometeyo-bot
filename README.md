# mixi2ほめてよbot

https://mixi.social/@hometeyo

ほめるんはまだ生まれたばかりで、いろんなことに興味津々。メンションすると、投稿の画像やテキストを元にほめるんがほめてくれます。会話にあったスタンプも押してくれます。

<img width="445" height="918" alt="ほめるんのスクリーンショット" src="https://github.com/user-attachments/assets/7af3bde2-9b82-4b7e-acb2-bbde6021828d" />

## 使い方

`@hometeyo` とメンションすると、ほめるんがあなたの投稿をほめにきます。画像やテキストに合わせて褒め言葉とスタンプを送ってくれます。
